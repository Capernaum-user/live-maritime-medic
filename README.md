# MDTS · M-Medic — 선박 응급의료 웹 데모

> 선박이라는 고위험 산업 현장의 응급 상황 대응을 위한 **엣지 AI 통합 솔루션(MDTS)** 의 웹 프런트엔드 데모.
> 스마트인재개발원 실전 프로젝트 (2026).

## 구성

- **React 18 + Vite** 기반 SPA, **Recharts** 실시간 차트
- 선박 내 센서(Jetson / Raspberry Pi) 데이터와 응급 상황 정보를 대시보드로 시각화

## 실행

```bash
npm install
npm run dev
```
